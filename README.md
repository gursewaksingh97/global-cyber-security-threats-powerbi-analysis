# 🛡️ Global Cyber Security Threats Analysis

## 📌 Project Overview

The Global Cyber Security Threats Analysis dashboard is a Power BI project designed to analyze cyber security incidents across multiple countries, industries, attack types, and attack sources. The dashboard provides valuable insights into affected users, financial losses, resolution efficiency, and country-wise cyber threat trends between 2015 and 2024.

This project helps identify major cyber security risks, evaluate their impact, and understand patterns in global cyber attacks through interactive visualizations and business intelligence techniques.

---

## 🎯 Project Objectives

* Analyze global cyber security incidents from 2015–2024.
* Identify the most common cyber attack types.
* Measure the impact of attacks on affected users.
* Evaluate financial losses caused by cyber threats.
* Analyze attack sources and targeted industries.
* Study incident resolution trends and defense mechanisms.
* Compare cyber security metrics across countries.
* Generate actionable insights through interactive dashboards.

## 🗂️ Dataset Information

The dataset contains cyber security incident records from 2015 to 2024 across multiple countries and industries.

### Dataset Features

* Country
* Year
* Attack Type
* Attack Source
* Targeted Industry
* Financial Loss
* Affected Users
* Resolution Time
* Defense Mechanism Used
* Security Vulnerability

* Total Affected Users
* Total Financial Loss
* Average Resolution Time
* Most Common Attack Type

* Country With Highest Financial Loss

## 🔍 Key Insights

* DDoS emerged as the most common attack type.
* Nation-State was identified as a major attack source.
* The UK recorded the highest financial losses.
* Brazil experienced the highest number of affected users.
* Average incident resolution time remained around 36 hours.
* IT and Banking sectors were among the most impacted industries.

---

## 📌 Conclusion

This dashboard provides a comprehensive view of global cyber security threats and their impact across countries and industries. The analysis helps understand attack patterns, assess financial risks, evaluate defense effectiveness, and support data-driven cyber security decision-making.
